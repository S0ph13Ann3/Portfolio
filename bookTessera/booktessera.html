project overview

BookTessera is a modern ticketing platform designed to combat the stress and unfairness of the typical online ticket purchasing experience. For years, fans and artists have been frustrated by scalpers, bots, and system crashes during high-demand sales. Our team set out to create a more equitable, transparent, and user-friendly solution that prioritizes real fans.

The challenge:

Anyone who has tried to buy tickets for a popular concert knows the pain. The process is often a frantic race against bots and other buyers, plagued by crashing websites, confusing queues, and a lack of transparency. Artists also suffer from the fan backlash that results from these negative experiences.

Our primary challenges were:
- Technical Implementation
- Fairness
- User Experience

The goal:

Our main goal was to design and develop a platform that offered an optimized general sales experience and an efficient purchase process.

To achieve this, we set the following objectives:

Enhanced User Management
A Fair Queuing System
Transparent Seat Allocation
Secure & Seamless Checkout

From design to code

Bringing the designs to life required a robust tech stack to handle the complex backend requirements while delivering a smooth frontend experience.

The system was designed to be scalable and responsive.

Event & User Data: We used Firestore, a NoSQL database, to store event information and user data. This allowed for real-time updates and flexible data structures.

Frontend Interface: The user interface was built with ReactJS and styled with Material-UI. This combination enabled us to create a dynamic, component-based, and visually consistent application.
