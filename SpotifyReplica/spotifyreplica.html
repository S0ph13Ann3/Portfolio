SPOTIFY REPLICA

View Prototype
Google Developer Student Club (GDSC) Duration: August 2022 Tool: Figma

project overview

The Spotify redesign project was an exercise I participated in as part of a "Figma 101" workshop hosted by the Google Developer Student Club (GDSC), where participants were tasked to replicate Spotify’s user flow and utilize their component library to create a more personalized UI, while learning how to replicate real-world design systems and apply consistent UI principles within Figma.

the challenge:

For this exercise, the main persona was ourselves—we were tasked with redesigning Spotify’s interface to reflect our own music habits and personalities. This brought the challenge of introducing personalization in a way that felt authentic and meaningful, while still working within Spotify’s existing structure and design system.

spotify's layout

In the workshop, we began by first analyzing Spotify's structure and layout to gain a clear understanding of how its interface components are organized, which helped inform the placement of elements within my own UI design.

components:

We were also given a component library that had the icons, colors, and typography needed to replicate Spotify’s UI.
